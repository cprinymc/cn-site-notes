# cn-site-notes

本项目用于归档和发布多个独立的 HTML 页面。页面内容涉及网站相关的笔记、记录与参考信息，不针对任何单个域名或特定网站。

## 项目简介

cn-site-notes 是一个基于 GitHub Pages 的静态文档归档仓库。仓库内包含若干独立 HTML 文件，每个文件对应一个独立的页面，用于保存与网站相关的各类笔记或记录。这些页面彼此独立，不构成完整的站点结构，适合作为轻量级的信息存储与分享方式。

## 目录说明

```
├── README.md          # 本说明文件
├── index.html         # 可选的主页（可能为空或引导页）
└── pages/             # 存放独立 HTML 页面的目录
    ├── page-001.html
    ├── page-002.html
    └── ...
```

- `pages/` 目录下存放所有独立的 HTML 页面文件。
- 页面文件名通常采用有序编号或简短描述，以方便识别。
- 仓库根目录可能包含一个简单的 `index.html`，仅用于占位或导航。

## 页面归档说明

- 每个 HTML 页面均为独立完整文档，包含自身的样式与内容。
- 页面之间无依赖关系，可单独访问或引用。
- 归档内容以网站相关的笔记、记录为主，不包含可执行脚本或动态数据。
- 页面可能引用外部资源（如 CSS、字体、图片），但尽量保持轻量。
